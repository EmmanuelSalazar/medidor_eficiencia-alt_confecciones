title: Instalación y configuración
---

En esta seccion, encontraremos los pasos para instalar y configurar el medidor de eficiencia en tu entorno local.

## Requisitos

- PHP 8.0 o superior (Recomendado 8.2)  
- MySQL 8.0 o superior (Recomendado 8.2)
- Node.js 18.0 o superior (Recomendado 20.11)
- Git (Opcional, para clonar el repositorio)

## Instalación

1. Clona el repositorio en tu máquina local, ya sea descargando el zip o ejecutando el siguiente comando:
:::warning[Opcional]
Si no tienes git instalado, puedes descargar el zip desde el boton de codigo y descomprimirlo en tu máquina local.
:::

```bash
git clone https://github.com/EmmanuelSalazar/medidor_eficiencia-alt_confecciones
```

2. Instala las dependencias del proyecto.
